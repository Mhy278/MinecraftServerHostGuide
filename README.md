# 简介
欢迎来到《我的世界服务器搭建指南》（MinecraftServerHostGuide）。

本教程面向第一次搭建我的世界Java版服务器的小白服主。

学习完本教程后，你将会掌握基础的我的世界服务器搭建技能。
